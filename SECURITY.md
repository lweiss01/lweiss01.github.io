# Security Policy

## Supported Versions

The following versions of this project are currently being supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please report it through the GitHub [Security Advisory](https://github.com/lweiss01/lweiss01.github.io/security/advisories/new) feature.

We take security seriously and will respond to your report as soon as possible. Please do not report security vulnerabilities through public GitHub issues.
